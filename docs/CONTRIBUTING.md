# Contributing to Free Games Claimer

Thank you for your interest in contributing to Free Games Claimer!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/free-games-claimer.git`
3. Create a feature branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Push to your fork: `git push origin feature/your-feature`
6. Create a Pull Request

## Development Setup

```bash
# Install dependencies
cd backend && npm install
cd ../frontend && npm install

# Run tests
npm test

# Start development
npm run dev
```

## Code Style

- Use ESLint configuration
- Follow existing code patterns
- Write meaningful commit messages
- Add tests for new features

## Pull Request Process

1. Update documentation
2. Add/update tests
3. Ensure all tests pass
4. Request review from maintainers
5. Address review comments

See [docs/GIT_COMMIT_INSTRUCTIONS.md](./GIT_COMMIT_INSTRUCTIONS.md) for detailed git workflow.

## Questions?

Create an issue or contact maintainers.
